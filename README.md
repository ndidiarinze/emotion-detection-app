# Final Project
## Emotion Detection Application using IBM Watson NLP
This project implements an Emotion Detection Application using IBM Watson Natural Language Processing.
The application analyzes text input and identifies emotions such as:
- anger
- disgust
- fear
- joy
- sadness

The project demonstrates:

- Building a Python emotion detection module
- Packaging Python code into a reusable package
- Implementing unit tests
- Creating a Flask web application
- Deploying the application and testing it through a web interface
